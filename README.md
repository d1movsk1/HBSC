# MK Youth Health Dashboard

Интерактивен веб-dashboard кој визуелизира HBSC податоци за физичка активност, спиење и здрави навики кај македонски адолесценти на возраст 11–15 години.

Изработен како дополнителна задача по предметот **ПНУВ** (Примена на компјутери во учење и воспитание), инспириран од наодите на меѓународната **HBSC студија (Health Behaviour in School-aged Children) 2022/23**.

---

## Преглед

Проектот адресира забележителен пад на физичката активност и квалитетот на сон кај македонски тинејџери, со цел да ги направи овие податоци достапни и разбирливи за широка публика.

![Dashboard preview](https://img.shields.io/badge/demo-отвори%20во%20прелистувач-1D9E75?style=flat-square)

---

## Функционалности

- **4 тематски секции** — Преглед, Физичка активност, Спиење, Навики
- **Филтер по пол** — динамична споредба момчиња / девојчиња
- **12 интерактивни графици** — bar, line, doughnut и horizontal bar charts
- **Метрик картички** со клучни статистики
- **„Знаеше ли?" факти** базирани на HBSC наоди
- **Dark mode поддршка** — автоматски според системски поставки
- **Responsive дизајн** — работи на мобилни и десктоп уреди

---

## Теми опфатени во dashboard-от

| Тема | Индикатори |
|------|-----------|
| Физичка активност | % активни 60+ мин/ден, тренд по возраст и пол, видови активности, пречки |
| Спиење | Часови сон vs СЗО препорака, квалитет на сон, причини за лош сон |
| Здрави навики | Исхрана, екранско време, споредба по возраст |

## Технологии

- **HTML5 / CSS3 / JavaScript** — без frameworks, без инсталација
- **[Chart.js 4.4](https://www.chartjs.org/)** — визуелизација на податоци
- **[Tabler Icons](https://tabler-icons.io/)** — иконографија
